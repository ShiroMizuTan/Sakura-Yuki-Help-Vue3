介绍 ​

Sakura Yuki是一个专为QQ客户端开发的娱乐兼功能性机器人。

触发方式 ​

Sakura Yuki大多指令是通过匹配消息开头、匹配消息末尾以及正则表达式等方法来识别发送的消息然后触发对应功能的。因此，你应当以一定规则发送信息。

TIP

你需要按照一定的规则来编写你的消息再发送，而不是当做类似ChatGPT、Deepseek等AI用自然语言的方式发送消息。
